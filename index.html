F1 Live Hub

JavaScript HTML5 CSS3 API AJAX JQuery

F1LiveHub é uma aplicação web que consome a API Ergast F1 para exibir informações atualizadas sobre a Fórmula 1, incluindo:
👨 Pilotos - Lista de pilotos com detalhes e estatísticas
🏎️ Equipes - Informações sobre todas as equipes da temporada
🏁 Corridas - Calendário completo da temporada com datas e resultados
🏆 Classificação - Tabelas de pontuação de pilotos e construtores

Observação: A API Ergast F1 é uma API pública que não requer autenticação. Usada de modo gratuíto, há uma limitação de 10 requisições por minuto. Caso tente acesssar e não funcione, tente novamente mais tarde.
